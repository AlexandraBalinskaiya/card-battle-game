# Card Battle Game

A simple JavaScript card game where the user competes against the computer.

## Live Demo

https://alexandrabalinskaiya.github.io/card-battle-game/

## Description

Card Battle Game is a small frontend practice project built with HTML, CSS and JavaScript.

The user enters a name and plays against the computer for three rounds. In each round, both the user and the computer receive a random card. Each card has its own value, and after three rounds the game compares the total score and shows the winner.

## Features

* random card generation
* player vs computer game logic
* three-round match system
* score calculation
* card image rendering
* final result message
* automatic game reset

## Technologies

* HTML5
* CSS3
* JavaScript

## How to Use

1. Open the live demo.
2. Enter your name.
3. Click the Generate button.
4. Play three rounds against the computer.
5. The game will calculate the total score and show the winner.
